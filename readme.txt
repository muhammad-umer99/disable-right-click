Disable right-click and text selection to prevent content copying on your website.

== Description ==
This plugin helps protect your website content by disabling right-clicking and text selection. It also allows you to show a custom message when users attempt to copy content.

== Features ==
* Disable right-clicking on the website.  
* Disable text selection and content copying.  
* Option to show or hide a custom alert message.  
* Customizable message text.  

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/disable-right-click/` directory.  
2. Activate the plugin through the 'Plugins' screen in WordPress.  
3. Configure settings via 'Settings' > 'Disable Right Click'.
